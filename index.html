Project Synopsis

Problem

Need to implement better security on halls of residences. Allow residents to easily access hall through main gate. Security personnel can allow vistors on hall and be alerted if the visitor hasn't signed out after 1 A.M.

Solution

An automated lock system that uses student IDs. Gives students quick access to enter/exit main gate, allow security personnel to keep track of how many persons are on hall. Alertssecurity of visitors who haven't signed out arter 1 A.M.

Progress

  • Scan IDs without the need for special cards being given to students
  • Visitor logs are viewable by security officer
  • Automatically scan ID without interaction from security guard
